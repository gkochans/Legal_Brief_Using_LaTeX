\section{Introduction}
This is an example of a legal brief in a format commonly used by attorneys in the United States. It is produced using the \LaTeX\ document preparation system. In this context, there are a variety of benefits to using \LaTeX\ to generate a brief in PDF format instead of using Microsoft Word: \\
\begin{enumerate}
\item Effortless alignment of line numbers.
\item Advanced typesetting (ligatures, hyphenation, etc.) for an improved reading experience.
\item Complete separation of content from formatting limits distractions while drafting.
\item Heading and signature block information is specified at the beginning of the file. The document template is automatically populated with case-specific information. \\ \end{enumerate}
To use the template to generate a final PDF, make sure that the PDF file \href{http://www.gregkochansky.com/assets/BriefBkgd.pdf}{BriefBkgd.pdf} is saved in the same folder as the template file. You can alter the format of BriefBkgd.pdf itself by editing the file \href{http://www.gregkochansky.com/assets/BriefBkgd.tex}{BriefBkgd.tex}. The entire package of files can also be downloaded from this \href{https://github.com/gkochans/Legal_Brief_Using_LaTeX}{GitHub repository}. To edit the template files, follow the same basic syntax rules described in the \href{http://www.gregkochansky.com/assets/Contract_Automation_Using_LaTeX.pdf}{Contract Automation Using \LaTeX} documentation.
